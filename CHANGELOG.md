## v1.0.0 (Day 1〜3 基本実装完了・初期リリース)
**【システム構築】**
- Next.js (App Router), Tailwind CSS を用いた画面構築。
- Zustand による `currentId` やフラグの状態管理。
- Howler.js によるBGM/SEの再生・フェード管理（`AudioManager.tsx`）。
- Framer Motion と CSS を用いた演出システム（画面揺れ、暗転、フラッシュ等）。

**【コンテンツ実装】**
- 約35ノードに渡る「午前2時のインターホン」本編シナリオ。
- Pythonでゼロから生成した完全著作権フリーのオリジナル音源（着信音、ドアノブ、打撃音、足音、環境音）の適用。
- 覗き穴と暗い部屋の背景画像の適用。
- 3種類のエンディング画面（BAD, TRUE, HIDDEN）の実装。
- GitHubへのソースコードPush。
