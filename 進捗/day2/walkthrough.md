# Day 2: 演出 + 音響 + シナリオ本編 実装完了報告

決定したタイトル**『午前2時のインターホン』**と、監視・ストーカー系の不気味なシナリオ（イメージA）に基づき、Day 2の全タスクを完了しました！

## 実装内容
1. **タイトル変更**
   - タイトル画面の表示を『午前2時のインターホン』に更新しました。
2. **演出・エフェクトの実装 (`components/EffectLayer.tsx`)**
   - シナリオの展開に合わせて `shake` (揺れ), `glitch` (ノイズズレ), `fadeBlack` (暗転), `flash` (赤フラッシュ) が自動で発動するように組み込みました。
3. **音響の組み込み (`components/AudioManager.tsx`)**
   - 最初のクリックを検知してオーディオコンテキストを初期化し、BGMのクロスフェードやSEの単発再生ができるようにしました。
   - `public/bgm`, `public/se` にダミーの音声ファイルを配置しています。
4. **シナリオ本編の実装 (`lib/scenario.ts`)**
   - スマホを通じた監視の恐怖、玄関での選択肢、ベランダからの逃亡など、約35ノードに渡るシナリオを書き切りました。
   - 選択によって Bad End, True End への分岐が機能します。（※Hidden Endのフラグも仕込んでいます）

## 確認方法
現在ローカルサーバーが起動中の場合（`npm run dev`）、`http://localhost:3000` をブラウザで開くと、新しいシナリオと演出（画面の揺れや暗転など）を通しでプレイできます。

> [!NOTE]
> 現在、効果音やBGMは「無音のダミーファイル」になっているため音は鳴りませんが、システム上は連携できています。
