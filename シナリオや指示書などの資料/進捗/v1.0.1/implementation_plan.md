# 実装計画 v1.0.1: エンディングバグ修正 + 隠しルート追加

## 概要
プレイテストで発見されたバグ修正と、シナリオに組み込まれていなかった隠しエンディングの実装。

## 変更内容

### バグ修正
- **エンディング直行バグ**: タイトル画面（`app/page.tsx`）を `'use client'` 化し、`useEffect` でマウント時に Zustand の `currentId` を `'start'` にリセットする処理を追加。

### コンテンツ追加
- **Hidden End シナリオノード追加** (`lib/scenario.ts`):
  - `ignore_3` ノードに「画像をよく見る」選択肢を追加し、`hidden_1` ルートへ分岐。
  - `hidden_1` 〜 `hidden_6` の6ノードを追加（窓ガラスの反射→返信→クローゼット着信の流れ）。
  - エンディング到達で `ending: 'hidden'` を設定。

## 対応ファイル
- `app/page.tsx`
- `lib/scenario.ts`
