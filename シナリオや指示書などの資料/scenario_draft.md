# ホラーゲーム タイトル・シナリオ案

指示書の要件（深夜の通知、ジワジワ系ホラー、エンディング3種）を満たすシナリオのドラフトです。
内容をご確認いただき、修正や希望があれば教えてください！

---

## 1. タイトル案

以下の3つから選ぶか、別のアイデアがあれば教えてください。
- **案A：『午前2時のインターホン』** （王道で状況がわかりやすい）
- **案B：『開けて』** （シンプルで不気味）
- **案C：『未登録の番号』** （スマホからの恐怖を強調）

---

## 2. シナリオ構成（ノードの流れ）

全25ノード程度を想定したフローです。

### 序盤：日常の違和感
深夜、自室のベッドでスマホを見ていると、未登録の番号からメッセージが届く。
**「開けて」**
ここで最初の選択肢が出ます。

- **選択肢A：「無視して寝る」** → 【中盤Aへ】
- **選択肢B：「玄関へ様子を見に行く」** → 【中盤Bへ】

### 中盤A（無視ルート）
既読をつけずに寝ようとするが、再びスマホが鳴る。
**「起きているのはわかってるよ。後ろを見て」**
- **選択肢：「振り返る」** → 振り返ると真っ暗。安堵して前を向くと… → **【Bad End: 添い寝】**
- **選択肢：「部屋の電気をつける」** → 電気をつけようとするが点かない。窓ガラスに映る「部屋の中にいる何か」に気づき、急いで外に逃げる。 → **【True End: 逃避行】**

### 中盤B（玄関ルート）
恐る恐る玄関へ向かう。ドアスコープを覗くが、外には誰もいない。
- **選択肢：「ドアを開ける」** → 誰もいない廊下。足元に古いお守りが落ちている。拾うとフラグON（※Hiddenルートの鍵）。そのまま部屋に戻る。 → 【合流ポイントへ】
- **選択肢：「そのまま部屋に戻る」** → 【合流ポイントへ】

### 合流ポイント（自室への帰還）
部屋に戻ると、先ほどまで自分のいたベッドの布団が、誰かが寝ているように膨らんでいる。
スマホが鳴る。**「おかえり」**

- **選択肢：「布団をめくる」**
  - お守りを持っていない場合：中から伸びてきた無数の手に引きずり込まれる。 → **【Bad End: 奈落】**
  - お守りを持っている場合（フラグON）：布団をめくると同時に激しいノイズと悲鳴。お守りが燃え尽き、怪異は消え去る。朝チュン。 → **【Hidden End: 祓魔】**
- **選択肢：「そのまま家から逃げ出す」** → 玄関から飛び出す。深夜のコンビニで朝を待つ。 → **【True End: 逃避行】**

---

## 3. エンディングまとめ
1. **【Bad End】**：怪異の誘導に乗り、部屋の中の「それ」に捕まってしまう結末。（複数パターンあり）
2. **【True End】**：違和感にいち早く気づき、自室を放棄して外に逃げ延びる結末。
3. **【Hidden End】**：特定の行動（玄関を開けて外を確認する等）でキーアイテムを入手し、怪異を撃退する結末。

---

この方向性でよろしければ、これらを `lib/scenario.ts` 用のデータ（各テキストや演出・効果音の指定を含む）として実装に落とし込みます。
